## IDEIA PRINCIPAL
Desenvolvimento de uma tela de login para um aplicativo destinado ao checkpoint 1 de <b>Mobile Development & IOT</b> e próximas atividades. A interface apresenta design clean e distribuição de elementos que promovem uma melhor experiência de usuário (UX).

## TELA DESENVOLVIDA
![Print screen]('./assets/image.png')

## COMPONENTE
`LoginInterface.tsx`

Dispõe de elementos da página de login, como título, campos de preenchimento de informações (input) – endereço de e-mail mais senha – além de botão para login. Ademais, botão de esquecimento de senha e criação de uma nova conta.

## REQUISITOS
- Componentização
- Indentação
- Nomenclatura de variáveis em formato [Camel Case](https://www.freecodecamp.org/news/snake-case-vs-camel-case-vs-pascal-case-vs-kebab-case-whats-the-difference/)
- Tipagem de variáveis
- Hierarquia de elementos visuais
- Padrão de estrutura de pastas
- Combinação de cores em palheta
- Heurísticas de Nielsen
- Responsividade mobile
